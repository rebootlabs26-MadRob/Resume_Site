Hardware Diagnostic Platform

2023 - Present

The Python/Tkinter interface provides real-time hardware fault detection during boot, then transitions to active system monitoring once the OS loads. Monitors CPU temperature, memory usage, disk health, and network activity in real-time. Integrated full system security scanning to detect malicious software, corrupt data, and integrity violations. Designed for rapid motherboard diagnostics, ongoing system health monitoring, and repair workflow optimization.

Technical Features:

  • Custom Tkinter interface with cyberpunk theming and adaptive scaling
  • Integrated security scanning (malware detection, file integrity checks)
  • Real-time system metrics (CPU temp, memory, disk I/O, network stats)
  • Optimized for 320x480 TFT displays and Raspberry Pi hardware
